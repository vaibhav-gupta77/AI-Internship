# House Price Predictor

This project is a Streamlit app that predicts house prices from basic inputs.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Cloud

1. Push this folder to GitHub.
2. Open Streamlit Cloud.
3. Create a new app and select this repository and the app.py file.
4. Streamlit Cloud will install requirements from requirements.txt automatically.
